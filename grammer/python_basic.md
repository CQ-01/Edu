# Python 기본 문법

## 변수
"="를 이용해 변수를 할당할 수 있다.
ex) x = 5

## 자료형
다음과 같은 자료형이 있다.
- 숫자 (integers, floats)
- 문자(Strings)
- 리스트
- 딕셔너리
- Booleans(T/F)

## 연산자
다음과 같은 수학/논리적 연산자가 있다.
- 덧셈 (+)
- 뺄셈 (-)
- 곱셈 (*)
- 제곱수 (**)
- 나눗셈 (/)
- 몫 (//)
- 나머지 (%)

## 조건문
- `if`
- `else`
- `for` 반복문

## 함수
`def` 키워드를 이용해 고유의 함수(재사용할 수 있는 코드 모음)를 만들 수 있다.

## 모듈
`import` 를 이용해 파이썬 코드들을 모아 모듈을 만들 수도 있고, 타인이 만든 모듈을 가져와 사용할 수도 있다.

## 예외 처리
`try` 와 `except` 를 이용해 에러발생이나 예외 시 동작을 설정할 수 있다.
